Define the Server

Path

Operation GET
The fields marked with * are required.

1. Enter Server URL

Enter the URL template of the server below. You can include server variables in the URL template with the following syntax:
http://{domain}.example.com:{port}/api

Server URL is required.

2. Configure Server Variables

Configure the server variables found in the server URL template entered in the section above. Changing the server URL template might change the server variables in this section.

The server's URL template does not contain any server variables. This section will only display server variables that are included in the server URL template entered in the above section.
